# Pico Pwm Controller

A PWM fan controller runs on raspberry-pico

## Build Project

1. Install [docker compose](https://docs.docker.com/compose/install/)

2. Build

    ```bash
    docker/docker-compose-compile.sh
    ```

3. Copy the build/release/pwm_controller.uf2 into RPI-RP2 drive
